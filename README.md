# 🎬 Netflix Movies Data Analysis

> A data analysis project using Python & Pandas to explore trends, ratings, and audience behavior in Netflix-style movie data.

---

## 📌 Project Overview
This project focuses on analyzing a Netflix movies dataset by importing data from a CSV file into Jupyter Notebook.  
The goal is to clean, explore, and extract meaningful insights using Pandas.

---

## 📊 Dataset Highlights
The dataset contains key information such as:
- Movie titles and descriptions  
- Popularity and ratings  
- Genre and language  
- Release dates  

---

## ⚙️ Approach

✔ Data Loading from CSV  
✔ Data Cleaning & Formatting  
✔ Exploratory Data Analysis (EDA)  
✔ Insight Generation  
✔ Visualization using Pandas  

---

## 🧠 Key Insights

- 📈 Recent movies show higher popularity trends  
- ⭐ Higher vote count often leads to stable ratings  
- 🎭 Drama & Crime genres perform strongly in ratings  
- 🌍 Majority of content is in English language  

---
